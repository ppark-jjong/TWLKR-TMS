1. 인수인계 기능 (handover.html)
1.1. 인수인계 목록 표시
탭 구조: 인수인계와 공지사항을 탭으로 구분하여 표시
테이블 구조: 인수인계 목록을 날짜, 제목, 작성자, 관련 주문, 상태 등의 컬럼으로 표시
정렬 기능: 날짜, 제목 등으로 정렬 가능
상태 표시: 읽음/안읽음 상태 시각적 표시
1.2. 인수인계 등록 및 관리
등록 폼: 제목, 내용, 관련 주문번호(선택), 우선순위 설정 기능
에디터: 서식있는 텍스트 작성을 위한 간단한 에디터 제공
파일 첨부: 이미지나 문서 첨부 기능 (미리보기 포함)
자동 저장: 임시 저장 기능으로 작성 중 내용 보존
템플릿: 자주 사용하는 인수인계 형식 템플릿 선택 기능
1.3. 인수인계 상세 보기
상세 내용: 모달 창으로 인수인계 전체 내용 표시
관련 주문 링크: 관련 주문이 있을 경우 해당 주문으로 이동 링크
히스토리: 수정 이력 및 조회 이력 표시
확인 기능: 인수인계 확인 처리 및 확인자 기록
댓글 기능: 인수인계에 대한 추가 의견이나 질문 작성 가능
1.4. 공지사항 관리
공지 목록: 중요도, 작성일, 제목, 작성자 표시
중요 공지: 중요 표시된 공지는 상단 고정 및 강조 표시
등록 기능: 공지사항 제목, 내용, 중요도, 게시 기간 설정
검색 기능: 제목, 내용, 작성자로 검색 가능
권한 관리: 작성자만 수정/삭제 가능한 권한 체크
1.5. 데이터 처리 흐름
데이터 로드: 페이지 로드 시 handover_data.json 파일에서 인수인계 및 공지사항 데이터 로드
필터링: 로드된 데이터를 사용자 필터에 따라 JavaScript로 필터링
렌더링: 필터링된 데이터를 테이블 형태로 DOM에 렌더링
상태 관리: 읽음 상태와 같은 사용자별 상태 정보는 localStorage에 저장
데이터 저장: 새로운 인수인계나 공지사항 작성 시 임시로 localStorage에 저장
1.6. 연동 기능
알림 연동: 새 인수인계 등록 시 관련 담당자에게 알림 표시
대시보드 연동: 중요 인수인계 및 공지는 대시보드 요약에도 표시
주문 연결: 특정 주문과 관련된 인수인계는 해당 주문 상세에서도 확인 가능
2. 시각화 기능 (visualization.html)
2.1. 배송 현황 대시보드
요약 카드: 총 주문수, 완료율, 평균 배송 시간 등 핵심 지표 카드 형태로 표시
상태별 차트: 대기, 진행, 완료, 취소, 이슈 등 상태별 주문 분포를 파이/도넛 차트로 시각화
시간대별 차트: 시간대에 따른 배송 상태 변화를 선 그래프로 표시
트렌드 분석: 날짜별 주문량과 완료율 추이를 복합 차트로 표시
2.2. 부서/창고별 분석
부서별 성과: CS, HES, LENOVO 등 부서별 배송 성과 비교 차트
창고별 현황: 서울, 부산, 광주, 대전 등 창고별 주문 처리 현황
히트맵: 지역별 배송 밀집도를 히트맵으로 시각화
성과 비교: 부서/창고 간 배송 완료율, 평균 처리 시간 등 성과 지표 비교
2.3. 세부 분석 도구
필터 패널: 날짜 범위, 부서, 창고, 배송 유형 등 다양한 필터 조합 제공
드릴다운: 차트의 특정 부분 클릭 시 세부 데이터로 드릴다운
마우스오버 상세: 차트 요소에 마우스 올릴 시 상세 데이터 툴팁 표시
데이터 테이블: 차트와 함께 상세 데이터를 테이블 형태로 함께 제공
2.4. 데이터 처리 및 시각화 기법
데이터 집계: 원시 주문 데이터를 통계적으로 집계하여 차트 데이터 생성
동적 업데이트: 필터 변경 시 실시간으로 차트 데이터 업데이트
애니메이션: 차트 전환 및 업데이트 시 시각적 애니메이션 효과
반응형 레이아웃: 화면 크기에 따라 차트 크기와 배치 자동 조정
2.5. 시각화 구현 기술
Chart.js 활용: 다양한 차트 유형(막대, 선, 파이, 도넛, 레이더 등) 구현
데이터 변환: dashboard_data.json의 원시 데이터를 차트에 적합한 형태로 가공
컬러 스키마: 데이터 유형과 상태에 따른 일관된 컬러 시스템 적용
범례 및 축: 직관적인 이해를 위한 범례, 축 레이블, 그리드 라인 적용
2.6. 인터랙션 및 사용자 경험
줌 기능: 특정 기간이나 영역 확대하여 상세 분석 가능
내보내기: 차트를 이미지(PNG, JPEG)나 PDF로 내보내기 기능
데이터 내보내기: 차트의 기반 데이터를 CSV, Excel 형식으로 내보내기
저장된 보기: 자주 사용하는 필터 조합을 저장하고 불러오기 기능
