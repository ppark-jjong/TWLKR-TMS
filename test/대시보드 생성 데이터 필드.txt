대시보드 생성 데이터 필드
생성 시각 관련 필드:
create_time - 대시보드 생성 시각 (DateTime, 필수)
모델 정의: create_time = Column(DateTime, nullable=False)
스키마: create_time: datetime

수정 시각 관련 필드:
update_at - 마지막 업데이트 시각 (DateTime, 자동 생성/갱신)
모델 정의: update_at = Column(DateTime, default=func.now(), onupdate=func.now(), nullable=False)
자동으로 현재 시간이 설정되며, 레코드가 업데이트될 때마다 갱신

수정 정보 관련 필드:
updated_by - 업데이트한 사용자 ID (생성자 정보도 포함)
모델 정의: updated_by = Column(String(50), nullable=True)
스키마: updated_by: Optional[str] = None

주요 시간 기록 필드:
depart_time - 출발 시간 (DateTime, 선택) => status가 대기에서 진행으로 바뀌는 순간의 시각
complete_time - 완료 시간 (DateTime, 선택) => status가 진행에서 완료,이슈로 바뀌는 순간
eta - 예상 도착 시간 (DateTime, 필수)

대시보드 생성 시 필수 필드:
class DashboardCreate(BaseModel):
    order_no: str            # 주문 번호
    type: DeliveryType       # 배송 유형 (DELIVERY/RETURN)
    department: Department    # 부서 (CS/HES/LENOVO)
    warehouse: Warehouse     # 창고 (SEOUL/BUSAN/GWANGJU/DAEJEON)
    sla: str                 # 서비스 수준 계약
    eta: datetime            # 예상 도착 시간
    postal_code: str         # 우편번호
    address: str             # 주소
    customer: str            # 고객명
    contact: Optional[str]   # 연락처 (선택)
    remark: Optional[str]    # 메모 (선택)



대시보드 업데이트 가능 필드:
class DashboardUpdate(BaseModel):
    order_no: Optional[str]           # 주문 번호
    type: Optional[DeliveryType]      # 배송 유형
    warehouse: Optional[Warehouse]    # 창고
    eta: Optional[datetime]           # 예상 도착 시간
    sla: Optional[str]                # 서비스 수준 계약
    postal_code: Optional[str]        # 우편번호
    address: Optional[str]            # 주소
    customer: Optional[str]           # 고객명
    contact: Optional[str]            # 연락처
    remark: Optional[str]             # 메모


생성 시각(create_time)은 자동으로 설정되며, 수정 시각(update_at)은 레코드가 업데이트될 때마다 자동으로 갱신됩니다. 수정한 사용자 정보는 updated_by 필드에 저장됩니다.
